Обзор

Измеритель ксчества воздуха AGAir сделан для измерения качества газа на предприятиях. Его можно использовать на фермах, для определения качества в загонах животных, или на заводах, для контроля вредности выбросов в атмосферу.
Проект разработан для предмета "Технология"
